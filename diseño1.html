📡 ¿QUE ES DISEÑO WEB?

Materiales, enlaces y herramientas útiles para estudiantes de redes

El Diseño Web es el proceso de planificar, crear y dar forma visual a sitios web. Involucra tanto la parte estética (cómo se ve) como la funcional (cómo se comporta e interactúa) de una página web.

¿Qué incluye el diseño web?
Diseño visual: colores, tipografías, imágenes, íconos, estructura visual.

  • Maquetación: organización del contenido en el sitio (menús, secciones, banners).

  • Interactividad: botones, formularios, animaciones, navegación intuitiva.

  • Adaptabilidad: que funcione bien en celulares, tablets y computadoras (diseño responsive).

  • Usabilidad y experiencia de usuario (UX/UI): que el sitio sea fácil y agradable de usar.


    • Objetivo del diseño web

    Hacer que un sitio sea atractivo, fácil de usar y eficaz para comunicar o vender algo.

    Ejemplo de diseño web bien hecho:

  • Página clara y bonita visualmente.

  • Menús intuitivos.

  • Rápida, segura y compatible con todos los dispositivos.
